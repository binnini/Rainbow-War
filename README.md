## 📌 요약

- **동기**: 기존 TPS 게임들보다 진입 장벽을 낮춰 모두가 즐길 수 있는 TPS 게임 개발
- **장르**: TPS, 멀티플레이
- **플랫폼**: Android
- **플레이 스타일**: 땅따먹기 + 슈팅 전투, 상대방보다 많은 영역 색칠하면 승리

## 📌 구현 내용

- Unity 5 엔진 사용
- 땅따먹기 알고리즘 구현
- Photon Server 프레임워크로 RPC 방식의 멀티 플레이 구현
- Plastic SCM(현 Unity Version Control)로 협업 환경 구축
